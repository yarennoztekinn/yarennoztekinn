## About Me

I'm a Software Engineering student at **Muğla Sıtkı Koçman University**, with a strong focus on **Data Science** and **Machine Learning**. I'm passionate about transforming raw data into meaningful insights and building end-to-end ML systems — from data collection and preprocessing all the way to model training and deployment.

My GitHub reflects my learning journey: personal projects, university work, and deep dives into algorithms, ML models, and software engineering principles.

## What I Work On

**Machine Learning & Deep Learning**
Built and trained models using **TensorFlow**, **Keras**, and **Scikit-learn**. Completed a full end-to-end ML project covering every stage from data processing and feature engineering to model deployment.

**Reinforcement Learning**
Designed a **Multi-Agent Reinforcement Learning** system using Tabular Q-learning, complete with a custom environment visualization built with Pygame.

**Computer Vision**
Contributing to a real-time image processing project using **OpenCV** and **MediaPipe**.

**Backend Development**
Developed backend systems using **Java/Spring Boot**, **Go/Fiber**, and **Python/Django** — with **PostgreSQL**, **Redis**, **JWT authentication**, and **Docker** containerization.

**DevOps & Infrastructure**
Experience with **Docker**, **Docker Compose**, and **Kubernetes** for container orchestration and scalable deployments.
